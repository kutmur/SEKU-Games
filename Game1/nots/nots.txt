📌 Vector2 Nedir?
Vector2, iki kayan noktalı sayıyı (float) temsil eden bir yapıdır:

csharp
Copy
Edit
Vector2 position = new Vector2(5.0f, 10.0f);
Bu örnekte:

position.x = 5.0

position.y = 10.0

🧠 Ne için kullanılır?

Yönleri ve hareketleri belirtmek

Mesafe veya hız hesaplamalarında

✅ Rectangle Nedir?
Rectangle, bir dikdörtgeni tanımlar: konumu ve boyutu ile birlikte.

csharp
Copy
Edit
using System.Drawing;

Rectangle rect = new Rectangle(x, y, width, height);
x ve y: Dikdörtgenin sol üst köşesinin koordinatları

width: Genişlik

height: Yükseklik

📌 Örnek Kullanım:
csharp
Copy
Edit
Rectangle myRect = new Rectangle(10, 20, 100, 50);

// Erişim:

int right = myRect.Right;   // 110
int bottom = myRect.Bottom; // 70

🎯 _spriteBatch.Begin(samplerState: SamplerState.PointClamp); Ne İşe Yarar?
Bu satır, SpriteBatch nesnesi ile 2D sprite çizimi başlatılırken, piksel tabanlı net görüntü elde etmek için "PointClamp" adı verilen bir örnekleme (sampling) ayarını kullanır.

🧱 Parçalarına Ayıralım:
🔹 _spriteBatch.Begin(...)
Bu metod, ekrana çizim yapılmaya başlanacağını belirtir.

Begin() çağrıldıktan sonra Draw(...) metoduyla sprite'lar çizilir.

En sonunda End() çağrılarak çizim kapatılır.

csharp
Copy
Edit
_spriteBatch.Begin(...);  // Çizime başla
_spriteBatch.Draw(...);   // Sprite çiz
_spriteBatch.End();       // Çizimi tamamla
🔹 samplerState: SamplerState.PointClamp
samplerState, texture’ların nasıl örnekleneceğini (görüntüleneceğini) belirler.

PointClamp, pikselleri "keskin" tutar. Yani ölçeklendirilen sprite'larda bulanıklaştırma yapılmaz.

Bu, özellikle pixel art (piksel sanatı) oyunları yaparken önemlidir. Görseller net kalır, bulanıklaşmaz.

🧪 Karşılaştırma:
SamplerState	Açıklama
PointClamp	Her piksel net kalır. Pixel art için idealdir.
LinearClamp	Pikseller arasında yumuşak geçiş yapılır (bulanıklaştırma)
AnisotropicClamp	Açılı dokular için daha kaliteli örnekleme

🎮 Basit Örnek Kullanım:
csharp
Copy
Edit
_spriteBatch.Begin(samplerState: SamplerState.PointClamp);
_spriteBatch.Draw(myTexture, new Vector2(100, 100), Color.White);
_spriteBatch.End();
